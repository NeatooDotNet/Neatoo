# Neatoo 10.6.2

**Release Date:** 2026-01-10
**Type:** Patch Release (Unit Testing Support)
**Breaking Changes:** None

---

## Summary

This release adds a parameterless constructor to `EntityBaseServices<T>` to simplify unit testing of entity classes without requiring dependency injection setup.

---

## What's New

### Parameterless Constructor for EntityBaseServices<T>

A new parameterless constructor enables creating `EntityBaseServices<T>` instances for unit testing without configuring a factory:

```csharp
// Unit test setup - no DI required
var services = new EntityBaseServices<MyEntity>();
```

**Important limitations:**
- Factory is set to `null`
- Save operations will fail
- For unit testing only - do not use in production code

This enables testing entity behavior (property changes, validation rules, state transitions) without the overhead of setting up the full factory infrastructure.

---

## API Changes

### New Constructor

| Class | Constructor | Purpose |
|-------|-------------|---------|
| `EntityBaseServices<T>` | `EntityBaseServices()` | Unit testing without factory |
