# Neatoo 0.14.2

**Release Date:** 2026-03-02
**Type:** Bug Fix
**Breaking Changes:** No

---

## Summary

Updates central `Microsoft.Extensions.DependencyInjection` package version to 10.0.3 to match the net10.0 target framework. The previous central version (9.0.5) was being resolved for net10.0 Blazor WASM consumers, causing "Could not resolve the signature of a virtual method" publish failures due to version mismatch between the dependency and the WASM Mono runtime.

---

## What's Fixed

- Central `Microsoft.Extensions.DependencyInjection` version in `Directory.Packages.props` updated from 9.0.5 to 10.0.3
