# Neatoo 0.14.0

**Release Date:** 2026-03-01
**Type:** Dependency Update
**Breaking Changes:** No

---

## Summary

Updates KnockOff and RemoteFactory dependencies to v0.x.x versions. The previous v10.x.x packages have been unlisted from NuGet and replaced with v0.x.x to reflect pre-1.0 maturity.

---

## Dependencies

| Package | Previous | Updated |
|---------|----------|---------|
| Neatoo.RemoteFactory | 10.14.0 | 0.16.0 |
| Neatoo.RemoteFactory.AspNetCore | 10.14.0 | 0.16.0 |
| KnockOff | 10.26.0 | 0.45.0 |

---

## Migration

No Neatoo API changes. If you reference KnockOff directly in your tests, note these API renames:

| Old API | New API |
|---------|---------|
| `stub.Prop.OnGet = () => value` | `stub.Prop.Get(() => value)` |
| `stub.Method.OnCall(cb)` (returns value) | `stub.Method.Return(cb)` |
| `stub.Method.OnCall(cb)` (void) | `stub.Method.Call(cb)` |
| `Times.Once` / `Times.Never` | `Called.Once` / `Called.Never` |
