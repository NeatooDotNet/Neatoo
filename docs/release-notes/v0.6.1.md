# Neatoo 0.6.1

**Release Date:** 2026-01-09
**Type:** Patch Release (Dependency Update)
**Breaking Changes:** None

---

## Summary

This release upgrades KnockOff from 10.11.1 to 10.12.0 and completes the migration from Moq to KnockOff for all test projects. Moq has been removed as a dependency from `Neatoo.UnitTest` and `Person.DomainModel.Tests`.

---

## What's New

### KnockOff 10.12.0 Upgrade

KnockOff 10.12.0 resolves all previously identified blockers for Neatoo interface stubbing:

| Feature | Status |
|---------|--------|
| Generic methods (`AddRule<T>`, `RunRule<T>`) | Now supported |
| Neatoo domain interfaces (`IPersonPhoneList`, etc.) | Now supported |
| BCL interface compatibility | 117 interfaces tested |

### Moq Removed from Test Projects

| Project | Status |
|---------|--------|
| `Neatoo.UnitTest` | Moq removed |
| `Person.DomainModel.Tests` | Moq.EntityFrameworkCore removed |
| `Neatoo.Samples.DomainModel.Tests` | Moq kept (documentation samples) |
